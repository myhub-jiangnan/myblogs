# git 常用命令

## git add .
>
> git add . 用于修改，添加后，保存文件状态
## git add --a 
> git add --a  可以用于保存包括删除文件的操作，

## git  branch 
> 查看当前分支

## git branch branchName
> 切换到某个分支
## git checkout -b branchName
>  创建并切换到新的分支

## git pull
> 从仓库拉取最新代码

## git push origin
> 推送代码到远程仓库主分支

## git push branchName
> 推送代码到远程仓库某个分支
